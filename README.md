# practicaMod3

Este repositorio contiene el desarrollo de una práctica orientada a la implementación y administración de una base de datos PostgreSQL utilizando Docker en un entorno de desarrollo basado en GitHub Codespaces. El proyecto tiene como finalidad demostrar el proceso completo de configuración de un contenedor, la creación automática de una base de datos y la ejecución de scripts SQL para la generación de tablas e inserción de datos iniciales.
Asimismo, se documenta la estructura del proyecto, incluyendo el uso de un archivo Dockerfile, la configuración mediante docker-compose.yaml y la organización de los scripts SQL necesarios para inicializar la base de datos. Una vez desplegado el contenedor, se realizan consultas SQL para verificar el correcto funcionamiento de las tablas y de la información almacenada.
Como parte de la práctica, también se integra la conexión entre PostgreSQL y Python mediante la biblioteca psycopg2, permitiendo establecer comunicación con la base de datos para realizar operaciones de acceso y gestión de información. El proyecto fue desarrollado con fines académicos para fortalecer conocimientos en tecnologías de contenedores, administración de bases de datos, control de versiones con Git y desarrollo colaborativo utilizando GitHub Codespaces
